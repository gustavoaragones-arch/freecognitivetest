Print-friendly instructions and scoring checklist for clock drawing sessions.

Intro

Use this page as a printable workflow for clock drawing assessment practice.

Educational Content

Ask the participant to draw a clock set to 11:10. Review face completeness, number placement, and hand placement. Keep printed records for longitudinal tracking.

FAQ

Can I print from mobile?

Yes, if your device supports wireless printing.

Is one failed drawing diagnostic?

No, one result should be followed by broader assessment.
