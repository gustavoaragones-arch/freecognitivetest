Memory Test Printable

Print-ready recall worksheet for caregivers and educational screening sessions.

Intro

Use printable lists and delayed recall prompts to monitor memory performance.

Educational Content

Present a word list, distract with another task, then ask for delayed recall. Track score and date on each sheet to observe trends and decide whether follow-up is needed.

FAQ

What delay is recommended?

A short delay such as 1-3 minutes is common in quick screening.

Can this be used weekly?

Yes, with different word lists each session.
